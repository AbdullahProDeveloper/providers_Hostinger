মনে রাখবেন

আপনার ওয়েবসাইটটি সকলের সাথে শেয়ার করার জন্য আপনাকে এটি অনলাইনে হোস্ট করতে হবে। XAMPP শুধুমাত্র লোকাল সার্ভারের জন্য, যা শুধুমাত্র আপনার নিজের কম্পিউটারে accessible।

ফাইল আপলোড করার পদ্ধতি

  1. আপনার হোস্টিং কন্ট্রোল প্যানেলে লগইন করুন (সাধারণত cPanel)
  2. File Manager এ ক্লিক করুন
  3. public_html বা www ফোল্ডারে যান
  4. আপনার সকল ওয়েবসাইট ফাইল আপলোড করুন
  5. একটি database তৈরি করুন এবং কনফিগার করুন
  6. config.php ফাইলটি এডিট করে database credentials আপডেট করুন

ডোমেইন নাম রেজিস্ট্রেশন

একটি ডোমেইন নাম রেজিস্ট্রেশন করার জন্য:

  1. Namecheap, GoDaddy, বা কোন স্থানীয় provider এ যান
  2. পছন্দের ডোমেইন নাম search করুন (যেমন: yourbusiness.com)
  3. ডোমেইন নামটি কিনুন
  4. ডোমেইন নামটি আপনার হোস্টিং এর nameserver এর সাথে link করুন
  5. নাম server পরিবর্তন করতে 24-48 ঘন্টা সময় লাগতে পারে
